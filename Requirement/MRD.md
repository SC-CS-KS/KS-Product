
# 什么是市场需求文档?
```md
是软件产品项目由“准备”阶段进入到“实施”阶段的第一文档，其作用就是某个软件产品进行市场层面的说明，
这个文档的质量好坏直接影响到产品项目的开展，并直接影响到公司产品战略意图的实现。
```
```md
一般在互联网企业中MRD都是由运营人员和产品设计人员共同制定完成的
```
## MRD vs PRD
```md
PRD 是一个“承上启下”的作用：
“向上”是对MRD内容的继承和发展，
“向下”是要把MRD中的内容技术化，向研发部门说明产品的功能和性能指标。

该文档一般是由产品设计人员来完成，也就是传统意义上的需求分析。
```
